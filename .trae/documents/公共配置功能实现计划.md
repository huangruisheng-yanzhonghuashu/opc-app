# 公共配置功能实现计划

## 一、需求分析

### 功能模块

#### 1. 资讯页banner配置
| 字段 | 类型 | 说明 |
|------|------|------|
| 图片 | VARCHAR(500) | Banner图片URL |
| 文章id/链接 | VARCHAR(500) | 支持两种：文章ID或外部链接 |

#### 2. 采集信息源配置
| 字段 | 类型 | 说明 |
|------|------|------|
| 关键词 | VARCHAR(255) | 采集关键词 |
| 链接 | VARCHAR(500) | 信息源链接 |

## 二、数据库设计

### 2.1 资讯页banner配置表 core_banner

```sql
-- ----------------------------
-- 资讯页banner配置表 core_banner
-- ----------------------------
DROP TABLE IF EXISTS core_banner;
CREATE TABLE core_banner (
    id                  BIGINT(20)      NOT NULL AUTO_INCREMENT    COMMENT 'BannerID',
    title               VARCHAR(255)    DEFAULT NULL               COMMENT '标题',
    image_url           VARCHAR(500)    NOT NULL                   COMMENT '图片URL',
    link_type           CHAR(1)         DEFAULT '1'                COMMENT '链接类型（1文章ID 2外部链接）',
    link_value          VARCHAR(500)    DEFAULT NULL               COMMENT '链接值（文章ID或外部链接）',
    sort_order          INT             DEFAULT 0                  COMMENT '排序（数字越小越靠前）',
    status              CHAR(1)         DEFAULT '0'                COMMENT '状态（0启用 1禁用）',
    create_by           VARCHAR(64)     DEFAULT ''                 COMMENT '创建者',
    create_time         DATETIME                                    COMMENT '创建时间',
    update_by           VARCHAR(64)     DEFAULT ''                 COMMENT '更新者',
    update_time         DATETIME                                    COMMENT '更新时间',
    remark              VARCHAR(500)    DEFAULT NULL               COMMENT '备注',
    PRIMARY KEY (id),
    KEY idx_status (status),
    KEY idx_sort_order (sort_order)
) ENGINE=InnoDB AUTO_INCREMENT=1 COMMENT='资讯页banner配置表';
```

### 2.2 采集信息源配置表 core_collect_source

```sql
-- ----------------------------
-- 采集信息源配置表 core_collect_source
-- ----------------------------
DROP TABLE IF EXISTS core_collect_source;
CREATE TABLE core_collect_source (
    id                  BIGINT(20)      NOT NULL AUTO_INCREMENT    COMMENT '配置ID',
    keyword             VARCHAR(255)    NOT NULL                   COMMENT '关键词',
    source_url          VARCHAR(500)    NOT NULL                   COMMENT '信息源链接',
    source_type         VARCHAR(50)     DEFAULT 'twitter'          COMMENT '来源类型（twitter/telegram/youtube等）',
    status              CHAR(1)         DEFAULT '0'                COMMENT '状态（0启用 1禁用）',
    create_by           VARCHAR(64)     DEFAULT ''                 COMMENT '创建者',
    create_time         DATETIME                                    COMMENT '创建时间',
    update_by           VARCHAR(64)     DEFAULT ''                 COMMENT '更新者',
    update_time         DATETIME                                    COMMENT '更新时间',
    remark              VARCHAR(500)    DEFAULT NULL               COMMENT '备注',
    PRIMARY KEY (id),
    KEY idx_status (status),
    KEY idx_keyword (keyword)
) ENGINE=InnoDB AUTO_INCREMENT=1 COMMENT='采集信息源配置表';
```

### 4.3 删除脚本

#### sql/core_public_config_menu_delete.sql - 公共配置菜单删除脚本
```sql
-- ----------------------------
-- 公共配置菜单删除脚本
-- ----------------------------

-- 删除公共配置相关菜单权限
DELETE FROM sys_role_menu WHERE menu_id IN (400, 401, 4010, 4011, 4012, 4013, 4014, 4015, 402, 4020, 4021, 4022, 4023, 4024, 4025);

-- 删除采集信息源按钮权限

-- 删除采集信息源菜单
DELETE FROM sys_menu WHERE menu_id = 402;

-- 删除Banner按钮权限
DELETE FROM sys_menu WHERE menu_id IN (4010, 4011, 4012, 4013, 4014, 4015);

-- 删除Banner菜单
DELETE FROM sys_menu WHERE menu_id = 401;

-- 删除公共配置目录
DELETE FROM sys_menu WHERE menu_id = 400;
```
