# 素材管理功能实现计划

## 一、项目结构分析

### 现有项目结构
- **前端** (opc-admin-ui): Vue3 + Element Plus + Vite
- **后端** (opc-*): 多模块 Maven 项目
  - opc-admin: Controller 层
  - opc-core: 核心业务 (Member)
  - opc-system: 系统业务
- **数据库**: MySQL，通过 sys_menu 表管理菜单权限

### 代码规范
- **命名规范**: `Core` + 实体名 (如 CoreMaterial)
- **路径规范**: `/core/material`
- **API 模式**: `listMaterial`, `getMaterial`, `addMaterial`, `updateMaterial`, `delMaterial`

## 二、需求分析

### 2.1 一期功能 (必须实现)

#### 统一素材管理界面（Tab页签设计）

**界面布局**：单一页面 `素材管理`，包含3个Tab页签

| Tab | 名称 | view_permission | 说明 |
|-----|------|-----------------|------|
| Tab1 | 一级用户素材 | 1 | 普通套餐会员可查看的素材 |
| Tab2 | 二级用户素材 | 2 | VIP套餐会员可查看的素材 |
| Tab3 | 三级用户素材 | 3 | 超级VIP套餐会员可查看的素材 |

**view_permission 与 会员套餐级别对应关系**：
- `view_permission = 1` 对应 `core_member.current_package_level = 1`（一级用户）
- `view_permission = 2` 对应 `core_member.current_package_level = 2`（二级用户）
- `view_permission = 3` 对应 `core_member.current_package_level = 3`（三级用户）

**权限规则**：
- 高级别套餐会员可以查看低级别套餐的素材
- 三级会员可查看 view_permission=1,2,3 的素材
- 二级会员可查看 view_permission=1,2 的素材
- 一级会员仅可查看 view_permission=1 的素材

**每个Tab下的功能**：
- 素材列表展示
- 操作：编辑、上下线、置顶/取消置顶、删除
- 新增素材

**素材详情**（统一）：
- 素材ID、标题、作者（注明出处）
- 总结（有就展示，没有就不展示）
- 正文、原链接、发布时间
- 查看权限选择（默认一级权限 1）
- 内容类型（纯文本/图文/视频）
- 会员展示图片

### 2.2 二期功能
- 超级VIP会员素材管理（活动相关）

## 三、数据库设计

### 3.1 统一素材表 core_material

```sql
-- ----------------------------
-- 素材表 core_material
-- ----------------------------
DROP TABLE IF EXISTS core_material;
CREATE TABLE core_material (
    id                  BIGINT(20)      NOT NULL AUTO_INCREMENT    COMMENT '素材ID',
    title               VARCHAR(255)    NOT NULL                   COMMENT '标题',
    author              VARCHAR(100)    DEFAULT NULL               COMMENT '作者（注明出处）',
    summary             TEXT            DEFAULT NULL               COMMENT '总结',
    content             LONGTEXT        DEFAULT NULL               COMMENT '正文',
    original_url        VARCHAR(500)    DEFAULT NULL               COMMENT '原链接',
    publish_time        DATETIME        DEFAULT NULL               COMMENT '发布时间',
    view_permission     INT             DEFAULT 1                  COMMENT '查看权限（1一级套餐 2二级套餐 3三级套餐）',
    content_type        VARCHAR(20)     DEFAULT 'text'             COMMENT '内容类型（text/image/video）',
    cover_image         VARCHAR(500)    DEFAULT NULL               COMMENT '会员展示图片',
    category            VARCHAR(50)     DEFAULT NULL               COMMENT '分类（normal/vip_column/vip/svip）',
    status              CHAR(1)         DEFAULT '0'                COMMENT '状态（0上线 1下线）',
    is_top              CHAR(1)         DEFAULT '0'                COMMENT '是否置顶（0否 1是）',
    source              VARCHAR(50)     DEFAULT NULL               COMMENT '来源（crawler爬取/manual手动）',
    create_by           VARCHAR(64)     DEFAULT ''                 COMMENT '创建者',
    create_time         DATETIME                                    COMMENT '创建时间',
    update_by           VARCHAR(64)     DEFAULT ''                 COMMENT '更新者',
    update_time         DATETIME                                    COMMENT '更新时间',
    remark              VARCHAR(500)    DEFAULT NULL               COMMENT '备注',
    PRIMARY KEY (id),
    KEY idx_status (status),
    KEY idx_is_top (is_top),
    KEY idx_view_permission (view_permission),
    KEY idx_category (category),
    KEY idx_publish_time (publish_time)
) ENGINE=InnoDB AUTO_INCREMENT=1 COMMENT='素材表';

-- 分类说明：
-- category = 'normal' 时为普通素材（一级用户可看）
-- category = 'vip_column' 时为大咖专栏（二级用户可看）
-- category = 'vip' 时为VIP素材（二级用户可看）
-- category = 'svip' 时为超级VIP素材（三级用户可看，二期）

-- 权限对应关系：
-- view_permission = 1 对应 core_member.current_package_level = 1（一级套餐）
-- view_permission = 2 对应 core_member.current_package_level = 2（二级套餐）
-- view_permission = 3 对应 core_member.current_package_level = 3（三级套餐）
```

### 5.1 API 接口

**api/core/material.js**

```javascript
import request from '@/utils/request'

// 查询素材列表
export function listMaterial(query) {
  return request({
    url: '/core/material/list',
    method: 'get',
    params: query
  })
}

// 获取素材详情
export function getMaterial(id) {
  return request({
    url: '/core/material/' + id,
    method: 'get'
  })
}

// 新增素材
export function addMaterial(data) {
  return request({
    url: '/core/material',
    method: 'post',
    data: data
  })
}

// 修改素材
export function updateMaterial(data) {
  return request({
    url: '/core/material',
    method: 'put',
    data: data
  })
}

// 删除素材
export function delMaterial(id) {
  return request({
    url: '/core/material/' + id,
    method: 'delete'
  })
}

// 修改状态
export function changeMaterialStatus(id, status) {
  return request({
    url: '/core/material/changeStatus',
    method: 'put',
    data: { id, status }
  })
}

// 修改置顶状态
export function changeMaterialTop(id, isTop) {
  return request({
    url: '/core/material/changeTop',
    method: 'put',
    data: { id, isTop }
  })
}
```
